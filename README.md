## Laravel Comics Layout

Creo un nuovo progetto Laravel.<br>
Creo un file di layout in cui inserire la struttura comune di tutte le pagine del sito web (head, body, ...) eventualmente includendo header e footer tramite due partials.<br>
Creo poi una rotta per visualizzare la lista di tutti i fumetti recuperati da un file inserito nella cartella config e aggiungo lo stile sfruttando Sass.
